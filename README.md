
# Maven Market - Power BI Dashboard

**Análisis completo ventas Maven Market** (cadena supermercados Canadá/México/EEUU).

## Dataset
- 2 años transacciones (1997-98)
- Clientes, productos, tiendas, devoluciones
- Fuente: Maven Analytics Udemy

## Features
- **Modelo estrella**: 7 tablas, DAX measures (Revenue $1.7M, Profit 59%)
- **KPIs**: Transacciones, Profit, Returns (0.99%)
- **Geo**: Mapa ciudades, Treemap drill país>estado>ciudad
- **Trend**: Revenue semanal 1998, Gauge vs target
- **Bookmarks**: Navegación Portland 1000 ventas
